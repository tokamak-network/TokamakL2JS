# Code of Conduct

This project expects respectful, constructive collaboration.

## Our Standards

- Be respectful and professional.
- Focus feedback on ideas, behavior, and code.
- Assume good intent and ask clarifying questions.
- Avoid harassment, personal attacks, and discriminatory language.

## Scope

This Code of Conduct applies to project spaces, including issues, pull requests, and review discussions.

## Enforcement

Maintainers may moderate discussions and remove content that violates these standards.

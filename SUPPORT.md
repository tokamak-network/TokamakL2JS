# Support

## Getting Help

Use GitHub Issues for:

- bug reports
- documentation gaps
- usage questions for TokamakL2JS APIs

Issue tracker:

- https://github.com/tokamak-network/TokamakL2JS/issues

## Before Opening an Issue

1. Check the README examples and API exports in `src/index.ts`.
2. Include environment details (Node.js version, package version, command used).
3. Provide a minimal reproducible example when possible.

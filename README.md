# TokamakL2JS

TokamakL2JS specifies a layer 2 protocol of Tokamak Network. It is a variant of [EthereumJS](https://github.com/ethereumjs/ethereumjs-monorepo).

All source code for TokamakL2JS is currently stored [here](https://github.com/tokamak-network/Tokamak-zk-EVM/tree/dev/packages/frontend/synthesizer/src/TokamakL2JS). It will be moved to this repository soon.

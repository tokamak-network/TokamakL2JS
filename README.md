# TokamakL2JS

TokamakL2JS is a TypeScript library for Tokamak Network L2 transactions and state handling.

## Getting started

```bash
npm install
npm run build
```

## Usage

```ts
import {
  createTokamakL2Tx,
  createTokamakL2StateManagerFromStateSnapshot,
  poseidon,
} from './dist/index.js';
```
